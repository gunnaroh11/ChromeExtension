InPersoned ChromeExtension
===============

Uses precompiled handlebars template.

## To build:

Prerequisites:

	* node.js
	* npm
	* handlebars.js 1.3.0 compiler

Install handlebars compiler with npm:

	npm install handlebars@1.3.0 -g
